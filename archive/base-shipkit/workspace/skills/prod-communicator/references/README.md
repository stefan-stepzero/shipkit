# References Folder

This folder contains reference materials for the prod-communicator skill.

## Included References

- **reference.md** - Stakeholder communication best practices for different audiences
- **examples.md** - Sample communications (investor, exec, team, customer)

## Add Your Own References

You can add any helpful resources to this folder:

- Your company's communication templates
- Successful pitch decks (PDFs)
- Investor one-pagers from successful raises
- Board deck examples
- Customer case studies or testimonials
- Email templates for different audiences
- Brand voice guidelines
- Presentation design guidelines
- Previous stakeholder communications that worked well

**Claude will reference any materials in this folder when using the skill.**

## How to Use

When running `/prod-communicator`, Claude can access all files in this folder to:
- Understand best practices for each communication type
- See examples for inspiration
- Match your company's tone and style
- Follow your organization's communication standards
- Adapt templates to your brand voice

Just drop files here - Claude knows to look!
