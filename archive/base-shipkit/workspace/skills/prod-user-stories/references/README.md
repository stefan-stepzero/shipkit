# References Folder

This folder contains reference materials for the prod-user-stories skill.

## Included References

- **reference.md** - User story best practices, INVEST principles, prioritization methods
- **examples.md** - Complete user story examples for B2B SaaS, B2C mobile, e-commerce, developer tools

## Add Your Own References

You can add any helpful resources to this folder:

- User story templates from your team
- Examples of well-written stories from past projects
- Prioritization frameworks your organization uses
- Acceptance criteria checklists
- Industry-specific user story patterns
- Research on user needs and requirements
- Product requirement documents (PRDs)

**Claude will reference any materials in this folder when using the skill.**

## How to Use

When running `/prod-user-stories`, Claude can access all files in this folder to:
- Understand user story best practices
- See examples for different product types
- Reference your team's specific templates and patterns
- Follow your organization's requirements format
- Learn from your past successful stories

Just drop files here - Claude knows to look!
