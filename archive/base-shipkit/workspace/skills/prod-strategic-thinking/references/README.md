# References Folder

This folder contains reference materials for the prod-strategic-thinking skill.

## Included References

- **reference.md** - Strategy frameworks and validation methods
- **examples.md** - Completed business canvas examples

## Add Your Own References

You can add any helpful resources to this folder:

- Business strategy books/articles (PDFs)
- Competitor analysis reports
- Market research data
- Industry trend reports
- Your organization's strategy templates
- Previous product strategies

**Claude will reference any materials in this folder when using the skill.**

## How to Use

When running `/prod-strategic-thinking`, Claude can access all files in this folder to:
- Understand strategy best practices
- See examples from similar companies
- Reference your market research
- Follow your organization's strategy framework

Just drop files here - Claude knows to look!
