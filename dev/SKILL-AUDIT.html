The Pattern

Killed Skills Share One Trait

Claude does them well without instruction. Debugging, documenting, implementing — these are native capabilities. Adding a skill adds overhead without adding value.

Kept Skills Share One Trait

They force human decisions to be explicit OR they create persistence Claude lacks. Vision, architecture decisions, domain models, institutional memory — these require human input or survive beyond a session.
