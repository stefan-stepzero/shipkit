## Adding Your Own References

## Quick Start

The dev-finish skill has three options:
1. **Merge locally** - Integrate code immediately
2. **Keep as-is** - Preserve branch for later
3. **Discard** - Delete branch permanently

See reference.md for complete workflows and patterns.
