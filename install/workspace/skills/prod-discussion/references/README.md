# References Folder

This folder contains reference materials for the prod-discussion skill.

## Included References

- **reference.md** - Trade-off thinking frameworks, decision-making models, cognitive biases to avoid
- **examples.md** - Practical discussion examples showing how to clarify questions, present options, and analyze trade-offs

## Add Your Own References

You can add any helpful resources to this folder:

- Decision-making frameworks from your organization
- Product strategy documents
- Past trade-off analyses and retrospectives
- Pricing strategy analyses
- Build-vs-buy decision templates
- Competitive analysis frameworks
- ROI calculation templates
- Prioritization frameworks (RICE, value/effort matrices)

**Claude will reference any materials in this folder when facilitating product discussions.**

## How to Use

When running `/prod-discussion` or when Claude detects a product question/challenge, all files in this folder will be used to:
- Clarify the real question behind surface-level asks
- Frame decisions as trade-offs with pros/cons
- Reference best practices from your organization
- Present options colored by industry best practices
- Apply decision-making frameworks appropriate to your context

Just drop files here - Claude knows to look!
