# References Folder

This folder contains reference materials for the prod-market-analysis skill.

## Included References

- **reference.md** - Market research methodology, web search strategies, source validation
- **examples.md** - Completed market analysis examples for B2C and B2B

## Add Your Own References

You can add any helpful resources to this folder:

- Previous market analyses from other products
- Industry analyst reports (PDFs from Gartner, Forrester, etc.)
- Competitive intelligence documents
- Market sizing spreadsheets
- Competitor tracking data
- Customer interview notes mentioning competitors
- Win/loss analysis reports
- Pricing research (historical pricing pages, competitor pricing sheets)

**Claude will reference any materials in this folder when using the skill.**

## How to Use

When running `/prod-market-analysis`, Claude can access all files in this folder to:
- Understand market research best practices
- Learn competitive analysis frameworks
- See examples for inspiration
- Reference your previous competitive research
- Use your industry-specific data

Just drop files here - Claude knows to look!

## Research Tips

**This skill is research-intensive.** Claude will use WebSearch to:
- Find competitors
- Gather pricing information
- Read customer reviews (G2, Capterra, Reddit)
- Research market trends
- Analyze feature sets

Make sure you have WebSearch available when running this skill.
