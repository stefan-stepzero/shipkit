# References Folder

This folder contains reference materials for the prod-personas skill.

## Included References

- **reference.md** - Extended guidance for creating personas
- **examples.md** - Concrete persona examples

## Add Your Own References

You can add any helpful resources to this folder:

- Research papers (PDFs)
- Interview transcripts
- Survey results
- Competitor persona examples
- Industry reports
- User research notes

**Claude will reference any materials in this folder when using the skill.**

## How to Use

When running `/prod-personas`, Claude can access all files in this folder to:
- Understand persona best practices
- See examples for inspiration
- Reference your custom research
- Follow your organization's persona templates

Just drop files here - Claude knows to look!
