# References Folder

This folder contains reference materials for the prod-interaction-design skill.

## Included References

- **reference.md** - Extended guidance for user journeys and interaction design
- **examples.md** - Concrete examples from different product types (B2B SaaS, B2C mobile, e-commerce, dev tools)

## Add Your Own References

You can add any helpful resources to this folder:

- UX research papers (PDFs)
- Journey maps from competitors
- Usability test recordings/notes
- Design system documentation
- Wireframes and prototypes
- User flow diagrams
- Analytics reports showing user behavior
- Screenshots of interaction patterns you like

**Claude will reference any materials in this folder when using the skill.**

## How to Use

When running `/prod-interaction-design`, Claude can access all files in this folder to:
- Understand interaction design best practices
- See examples for different product types and stages
- Reference your custom research and user testing
- Follow your organization's design patterns
- Learn from your existing design system

Just drop files here - Claude knows to look!
