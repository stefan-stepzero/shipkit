## Template Variables

Replace these placeholders when generating HTML:

- `{Report Title}` - Main title (e.g., "Architecture Decisions")
- `{Report Subtitle}` - Subtitle (e.g., "Visual Decision Timeline")
- `{Current DateTime}` - Full timestamp (e.g., "December 29, 2025 at 2:30 PM")
- `{Current Date}` - Date only (e.g., "December 29, 2025")
- `{CONVERTED MARKDOWN CONTENT HERE}` - HTML-converted markdown content
