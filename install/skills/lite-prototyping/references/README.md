# lite-prototyping References

## Purpose
Extended documentation and examples for lite-prototyping skill.

## Contents

### examples.md
Complete examples, tips, and scenarios for rapid prototyping:
- **Tips for Effective Prototyping** - Best practices (mobile-first, real content, focus on one interaction)
- **Common Scenarios** - Validating flows, comparing approaches, importing mockups
- **Example Prototype Structure** - Complete React + Tailwind template with:
  - Share button interaction
  - Toast notifications
  - State management
  - Mock data
- **Template Variations** - Multi-step flows, forms with validation, lists with filtering

## When to Reference
Refer to `examples.md` when:
- Creating your first prototype (use the template as starting point)
- Stuck on how to structure a specific interaction
- Need tips for effective prototyping workflow
- Want to see complete working examples

## Quick Reference

**Prototyping workflow:**
1. User describes UI concept
2. Generate single-file HTML prototype (React + Tailwind via CDN)
3. User opens in browser
4. Iterate live with user watching
5. Extract learnings to spec via `/lite-prototype-to-spec`
