How Does It Work?

Session Hook Enforcement

SessionStart hook auto-loads /shipkit-master meta-skill. Skills are mandatory when applicable - no more "I'll do it later".

File Protection

settings.json makes outputs read-only. To update, you must re-run the skill. This prevents bypassing workflows.

Script-Driven Consistency

Claude uses templates and scripts - never creates files manually. Same format every time.

Unified Workspace

Everything lives in .shipkit/ - no project root pollution. Clean, organized, and version-controllable.
