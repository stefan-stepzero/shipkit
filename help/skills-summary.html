Framework Overview

ShipKit combines ProdKit (product discovery) and DevKit (technical development) into a complete workflow from initial idea to shipped code.

Complete Workflow:

ProdKit Discovery → /constitution-builder → DevKit Specs → /implement → Shipped Code

ProdKit (11 skills)

Product discovery and strategy skills. Define what to build and why.

Sequential Skills (9)

Run in order from strategy to success metrics:

/strategic-thinkingStep 1

/personasStep 2

/market-analysisStep 4

/brand-guidelinesStep 5

Agent Personas (5)

Specialized behaviors loaded by skills. Each skill specifies which agent persona to adopt.

Agent Role Used By
Discovery Agent Product discovery, user research, strategic thinking All ProdKit skills, /brainstorming
Architect Agent Technical planning, specifications, architecture /constitution-builder, /specify, /plan, /tasks
Implementer Agent TDD-focused coding, minimal implementation /implement, development tasks
Reviewer Agent Two-stage review: spec compliance + code quality /implement review stages, code reviews
Researcher Agent Deep research, exploration, cross-referencing Research phases, /brainstorming deep dives
