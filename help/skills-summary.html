Framework Overview

ShipKit is a unified framework with 34 Claude Code skills organized by prefix: prod-* (product discovery), dev-* (technical development), and any-* (cross-context). All skills live in a flat .claude/skills/ directory.

Complete Workflow:

prod-* Discovery → /dev-constitution-builder → dev-* Specs → /dev-implement → Shipped Code

Product Discovery Skills (11 prod-* skills)

Product discovery and strategy skills. Define what to build and why.

Sequential Skills (9)

Run in order from strategy to success metrics:

/prod-strategic-thinkingStep 1
Define business strategy, value proposition, and market positioning
/prod-personasStep 2
Identify and characterize target users with demographics and goals
/prod-jobs-to-be-doneStep 3
Map current state workflows and user pain points
/prod-market-analysisStep 4
Analyze competition using Porter's Five Forces
/prod-brand-guidelinesStep 5
Define visual direction, colors, typography, and brand voice
/prod-interaction-designStep 6
Design future state user journeys and interactions
/prod-user-storiesStep 7
Write requirements with MoSCoW prioritization
/prod-assumptions-and-risksStep 8
Identify strategic risks and assumptions to validate
/prod-success-metricsStep 9
Define KPIs, instrumentation, and measurement strategy
